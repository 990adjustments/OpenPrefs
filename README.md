OpenPrefs
=========

Description
-----------

OpenPrefs is a [Cinema 4D](http://www.maxon.net) script that
offers a quick way to open the Cinema 4D user preferences folder
without having to go through the Preferences dialog. Make sure to
make it a button on your shelf!

Installation & Usage
--------------------

Just place script and icon in your scripts directory.
